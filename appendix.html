總複習：習題與解答

題目 1

你想查看目前資料夾裡有哪些檔案，應該輸入什麼？

答案

ls

解析

這是 Linux 最常用的查看指令之一，幾乎每次移動到新資料夾都會用到。

題目 2

你想從 Linux 進入 Windows 的 C 槽使用者目錄，應該走哪個路徑？

答案

/mnt/c/Users

解析

這題在確認你是否理解 WSL 路徑映射。Windows 的磁碟會出現在 /mnt 下。

題目 3

如果腳本檔存在，但執行 ./hello.sh 失敗，第一個該檢查的是什麼？

答案

檢查檔案是否有執行權限，例如是否已執行 chmod +x hello.sh。

解析

很多初學者以為內容正確就能跑，但 Linux 還會檢查權限。這是腳本失敗很常見的原因。
