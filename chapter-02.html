Chapter 02

一定要會的基本 Linux 指令

先把日常移動、建立、查看和整理檔案的能力建立起來，Linux 才會從陌生變成可用。

ls

列出目前資料夾中的內容。

cd

切換資料夾，例如 cd /mnt/c/Users。

pwd

顯示目前所在路徑。

mkdir

建立資料夾。

cp

複製檔案或資料夾。

mv

移動檔案，或直接重新命名。

rm

刪除檔案，執行前要確認目標。

cat

快速查看文字檔內容。

Ubuntu 套件管理：你會最常碰到的兩個動作

sudo apt update
sudo apt install git
sudo apt install python3

sudo 可以理解為暫時取得管理員權限，apt 則是 Ubuntu 的套件管理工具。

一個很重要的小習慣

每執行完一個指令，都看一下結果再繼續。Linux 新手最容易卡住的地方，不是不會打指令，而是沒有觀察輸出。

習題與解答

題目 1

如果你想知道自己目前所在的完整路徑，應該使用哪個指令？

答案

pwd

解析

pwd 的意思是 print working directory，用來顯示目前工作目錄。它是確認位置最直接的指令。

題目 2

你要建立一個名稱為 project-a 的資料夾，應該輸入什麼？

答案

mkdir project-a

解析

mkdir 是 make directory。看到要「建立資料夾」，幾乎就該先想到它。

題目 3

sudo apt update 和 sudo apt install git 有什麼差別？

答案

前者更新套件索引，後者安裝 Git。

解析

update 是同步可安裝套件的最新資訊，不是升級軟體本身。install 才是實際把某個工具裝進系統。
